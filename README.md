# 绩点查询与后悔

一个本地可用的 CSV 绩点查询工具，支持上传、筛选、手动切换计入与结果下载。

## 适合谁使用

- 想快速核算当前绩点的同学
- 想模拟“如果这门课不计入会怎样”的同学
- 想用统一 CSV 模板管理课程成绩的同学

## 软件功能

- 初始为空面板，先上传 CSV 才开始计算
- 支持拖拽上传与手动选择上传
- 按学期筛选
- 每门课可手动切换“计入/不计入”
- 一键恢复默认计入状态
- 按学分加权计算平均绩点
- 下载当前结果为 CSV

## 快速开始（Windows）

### 方法 1：直接运行 exe（推荐）

1. 下载并打开 `GPAQueryRegret.exe`
2. 软件会自动打开浏览器页面（默认 `http://127.0.0.1:8000`）
3. 上传你的 CSV，开始计算

### 方法 2：脚本启动

1. 双击 `start_web.bat`
2. 浏览器打开后上传 CSV 使用

## 使用流程

1. 上传 CSV 文件
2. 选择要计算的学期
3. 在课程表里勾选/取消“计入”
4. 点击“计算绩点”刷新结果
5. 如需撤销手动修改，点“恢复初始状态”
6. 如需导出，点“下载当前结果CSV”

## CSV 格式要求

### 必须列（列名必须完全一致）

- 课程名称
- 修读时间
- 学分
- 等第
- 绩点

### 可选列

- 课程类型
- 备注

可选列可以没有；如果有，会在表格中展示。

## 等第与绩点规则

允许的等第：

- A+, A, A-, B+, B, B-, C+, C, C-, D, D-, F, P, NP

对应规则：

- A+, A：绩点必须为 4
- A-：3.7 到 3.8
- B+：3.3 到 3.6
- B：3.0 到 3.2
- B-：2.7 到 2.9
- C+：2.3 到 2.6
- C：2.0 到 2.2
- C-：1.7 到 1.9
- D：必须为 1.3
- D-：必须为 1.0
- F、P、NP：不计入绩点（可视作通过不计入或待补考）

额外校验：

- 学分必须大于 0
- 修读时间不能为空
- 非 F/P/NP 的课程，绩点不能为空

## 等第边框颜色

- A、A+、P：红色
- A-：橙色
- B+：黄色
- B：淡绿色
- B-：深绿色
- C+：蓝色
- C：深蓝色
- C-：紫色
- D：深紫色
- D-、F、NP：灰色

## 常见问题

1. 上传后提示“列名错误”
请检查表头是否与“必须列”完全一致（包括中文名称）。

2. 为什么 F/NP/P 不参与平均绩点
本软件将这三类视为不计入（F/NP 可理解为待补考状态）。

3. 为什么勾选后结果没变化
勾选只是修改预设，需要再点一次“计算绩点”才会刷新结果。
